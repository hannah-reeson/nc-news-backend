NorthCoders News




GET /api/topics/:topic_id/articles
Return all the articles for a topic using the topic ID

POST /api/topics/:topic_id/articles
Adds a new article to a topic using the topic ID

GET /api/articles/:article_id
Gets an article by its ID

GET /api/articles/:article_id/comments
Gets all the comments for a individual article

POST /api/articles/:article_id/comments
Adds a new comment to an article.

PATCH /api/articles/:article_id
Increases or Decreases the votes by 1 using article ID.

GET /api/comments/:comment_id
Returns comments from there ID

PATCH /api/comments/:comment_id
Increases or Decreases the votes by 1 using comment ID.

DELETE /api/comments/:comment_id
Deletes a comment

GET /api/users/:username
Returns user profile from there username.
